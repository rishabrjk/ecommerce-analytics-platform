# Cloud-Based E-Commerce Analytics Platform

## 🚀 Overview
Designed and built a scalable cloud-based analytics platform on Google Cloud Platform to process and analyze 100K+ e-commerce transactions. The system enables data-driven decision-making through customer segmentation and automated dashboards.

## 📊 Key Results
- Processed 100K+ transaction records using BigQuery
- Identified 4 customer segments using SQL window functions
- Detected a 22% churn-risk user group
- Reduced manual reporting effort by ~60%

## 🛠 Tech Stack
- Google Cloud Platform (BigQuery, Cloud Run)
- SQL
- Python

## 🏗 Architecture
Data Pipeline:
Raw Data → BigQuery → SQL Transformations → Cloud Run → Dashboard

## 📈 Key Features
- Serverless data pipeline
- Advanced SQL analytics (window functions, aggregations)
- Customer segmentation
- Business insights dashboard

## ⚙️ How to Run
1. Upload dataset to BigQuery
2. Execute SQL queries from `/sql` folder
3. Deploy Cloud Run service
4. Connect to dashboard
